Requirements
------------

- Be written in Java
- Must be a console application
- Have persistence
- Demonstrate separation of concerns
- Include at minimum 2 class objects/ models
- Application should run until the user decides to exit the application
- Handle errors / exceptions gracefully
- Persisted data must include multiple types of objects from your application
- Persisted data must include one many-to-many relationship
- If in SQL, data must be normalized to 3NF
- Save project to a public Git Repo
- Application must include unit testing
    - At least 10 individual tests
    - Mock the repo layer
    - Should have happy path, negative result tests, and edge cases
    - Apply TDD for one method
